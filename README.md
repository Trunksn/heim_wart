# HeimWart

Die Software HeimWart ist eine Desktopanwendung in Python, die Haushaltsgeräte, Anlagen und Fahrzeuge verwaltet. Sie protokolliert Wartungen und Reparaturen getrennt, berechnet daraus dynamisch die nächsten Fälligkeitstermine und warnt rechtzeitig vor überfälligen Maßnahmen, um so den Werterhalt zu sichern und kostspielige Folgeschäden zu vermeiden.

Modul: "Projekt: Software Engineering"

## Abhängigkeiten

Es wird eine [Python 3](https://www.python.org/downloads/) Installation benötigt.

Wenn die Software auf Linux ausgeführt wird, muss u. U. das Softwarepaket python3-tkinter / python-tk über die jeweilige Linux-Paketverwaltung installiert werden.

## Ausführung

Das Programm kann auf Microsoft Windows gestartet werden, indem ein Doppelklick auf die main.py oder der folgende Shell-Befehl durchgeführt wird:

```
py main.py
```
